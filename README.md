## 📜 Descrição

Este projeto busca automatizar a classificação de grãos de trigo utilizando aprendizado de máquina, aplicando a metodologia **CRISP-DM**. O objetivo é substituir o processo manual, sujeito a erros e demorado, por um sistema eficiente, preciso e escalável.


## 📁 Estrutura de pastas

Dentre os arquivos e pastas presentes na raiz do projeto, definem-se:

- <b>assets</b>: aqui está a imagem da logo da fiap utilizada no readme.

- <b>src</b>: Arquivo ipynb contendo o código do projeto e dataset utilizado.

- <b>README.md</b>: arquivo que serve como guia e explicação geral sobre o projeto (o mesmo que você está lendo agora).

## 🔧 Como executar o código

- Instalar as bibliotecas necessárias.
- Utilizar o dataset disponibilizado.
- Rodar o arquivo.
